### 풀이 언어 : C++

문제 구분 : #반복문
#### [LINK - 백준 2438 별 찍기 - 1](https://www.acmicpc.net/problem/2438)

### 문제
<hr>

첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 찍는 문제

### 입력
<hr>

첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.
### 출력
<hr>

첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.
### 풀이 이야기
<hr>

기본적인 반복문을 활용하여 출력하는 문제이다.

### 풀이 코드
<hr>

``` c++
#include <iostream>
 
int main(void) {
	int N;
	std::cin >> N;
	for (int i = 1; i <= N; i++) {
		for (int j = 1; j <= i; j++) {
			std::cout << "*";
		}
		std::cout << "\n";
	}
}
```
