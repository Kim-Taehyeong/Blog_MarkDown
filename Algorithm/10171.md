### 문제
<hr>


아래 예제와 같이 고양이를 출력하시오.
### 입력
<hr>

없음.
### 출력
<hr>

고양이를 출력한다.
### 풀이 이야기
<hr>

입출력을 통해서 결과를 출력하면 된다. 특수문자의 경우 `\` 역슬래쉬를 활용해서 출력하면 원하는 그림을 출력할 수 있다.

### 풀이 코드
<hr>

``` c++
#include <iostream>

int main(void) {
    std::cout << "\\    /\\\n";
    std::cout << " )  ( ')\n";
    std::cout << "(  /  )\n";
    std::cout << " \\(__)|\n";
}
```
