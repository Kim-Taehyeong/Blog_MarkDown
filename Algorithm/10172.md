기본적인 출력 문제이다. `\`기호를 잘 활용해서 특수 문자들을 출력하면 된다.

### 풀이 코드
<hr>

#include <iostream>

int main(void) {

	std::cout << "( 0 )\"\"\"\\\n";
	std::cout << "|\"^\"`    |\n";
	std::cout << "||_/=\\\\__|\n";
}
